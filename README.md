# Financial Indicators Dashboard

Real-time macroeconomic indicators and market valuation metrics powered by FRED and other financial APIs.

## Performance Optimizations
- Page load time: 30-40s → 4s (90% improvement)
- Treasury API: 7.6s → 1s (87% improvement)
- Optimized data ranges (last 10 years)
- Removed Yahoo Finance dependencies
- Manual refresh only (no auto-refresh)

## Live Demo
Visit: https://financial-dashboard-nu-three.vercel.app

## Tech Stack
- Next.js 14
- React Query
- Tailwind CSS
- Recharts
- FRED API
- Upstash Redis
